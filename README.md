## 네이밍 규칙

### id, 변수명
- Camel Case 사용
```js

// bad case
const UserName = 'James';

// good case
const userName = 'James';

```
```html

// bad case
<div id="UserName"></div>

// good case
<div id="userName"></div>

```
### 클래스
- kebab-case 사용

```html
// bad case
<div class="userName"></div>

// good case
<div class="user-name"> </div>
```
